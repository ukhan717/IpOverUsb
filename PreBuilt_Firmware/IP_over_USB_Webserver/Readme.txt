ReadMe.txt for STM32F746-DISCO_Eval prebuilt binary

STM32F746-DISCO_Eval.srec is a prebuilt binary file of the
application to run on a STM32F746-DISCO_Eval board.
It is prepared for download into internal flash
using J-Link without the need for an IDE.


How to run the prebuilt application:
====================================
1. Connect your target to your host PC via J-Link.
2. Execute StartBinary.bat, this can be run under Windows, Linux and Mac.
   On Linux use "bash StartBinary.bat"
